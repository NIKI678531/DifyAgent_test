## 技术指标计算流程

```
价格数据序列 (价格数组 P)
        │
        ├─────────────────┬──────────────┬──────────────┐
        │                 │              │              │
    ┌───▼────┐       ┌────▼──┐    ┌─────▼──┐    ┌─────▼──┐
    │ MA计算 │       │RSI计算 │    │波动率  │    │统计值  │
    └───┬────┘       └────┬──┘    └─────┬──┘    └─────┬──┘
        │                 │             │             │
        ├─────────────┐   │             │             │
        │             │   │             │             │
    ┌───▼────┐   ┌────▼──┐   ┌─────┐   │        ┌────▼──┐
    │P[n-5:] │   │涨跌值 │   │收益率│   │        │Max(P) │
    │平均    │   │rs    │   │log  │   │        │Min(P) │
    │= MA5   │   │=(up) │   │%    │   │        │Mean(P)│
    └───┬────┘   │/(down)
        │        └────┬──┘   └─────▼───┐      └────┬──┘
        │             │               │            │
    ┌───▼────┐   ┌────▼──┐   ┌────────▼───┐   ┌────▼──┐
    │P[n-10:]│   │RSI=   │   │Std×√252    │   │高/低  │
    │平均    │   │100-   │   │×100=Vol%   │   │差    │
    │= MA10  │   │100/(1 │   │            │   │      │
    │        │   │+rs)   │   │            │   │      │
    └───┬────┘   └────┬──┘   └────────┬───┘   └────┬──┘
        │             │               │            │
    ┌───▼────┐   ┌────▼──┐   ┌────────▼───┐   ┌────▼──┐
    │P[n-20:]│   │RSI值  │   │年化波动率  │   │High   │
    │平均    │   │(0-100)│   │            │   │标准化 │
    │= MA20  │   │       │   │            │   │      │
    └────┬───┘   └───────┘   └────────────┘   └────┬──┘
         │                                     ┌────▼──┐
         │                                     │Low    │
         │                                     │Ave    │
         │                                     └───────┘
         │
    ┌────┴───────────────────────────────────────┐
    │      趋势判断 (determine_trend)             │
    │                                             │
    │  MA 排列关系:                              │
    │  P > MA5 > MA10 > MA20  → 强势上涨         │
    │  P > MA5 > MA20 > MA10  → 上涨            │
    │  P < MA5 < MA10 < MA20  → 强势下跌         │
    │  P < MA5 < MA20 > MA10  → 下跌            │
    │  其他情况 → 震荡                          │
    └────┬───────────────────────────────────────┘
         │
    ┌────▼──────────────────────────────────────────────┐
    │      建议生成 (generate_recommendation)            │
    │                                                    │
    │  规则优先级:                                      │
    │  Level 1: 极端信号(RSI>70 or RSI<30)            │
    │  Level 2: 强势趋势(强势上/下跌)                 │
    │  Level 3: 短期快速变动(涨跌>5%)                │
    │  Level 4: 中性状态(震荡)                       │
    │                                                    │
    │  输出: 详细建议文本 + 理由 + 操作建议 + 关键位置  │
    └────┬──────────────────────────────────────────────┘
         │
    ┌────▼──────────────────┐
    │    响应数据构造         │
    │   (StockAnalysis)      │
    │                         │
    │   • stock_code         │
    │   • stock_name         │
    │   • current_price      │
    │   • ma5, ma10, ma20    │
    │   • rsi, volatility    │
    │   • highest/lowest     │
    │   • trend              │
    │   • recommendation     │
    │   • currency           │
    │   • analysis_time      │
    └────┬──────────────────┘
         │
    ┌────▼──────────────┐
    │   JSON 序列化      │
    │   HTTP 200 OK     │
    └────┬──────────────┘
         │
    返送客户端
```

## 市场数据源映射

```
A股 → akshare.stock_zh_a_hist() → 前复权历史数据
美股 → 多层次备选 (实时→延迟→示例)
港股 → akshare.stock_hk_hist() → 复权历史数据
指数 → akshare.stock_zh_index_daily() → 指数行情

所有数据 → standardize_dataframe() → 统一格式
```
